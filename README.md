# movie_review

Сайт кинобиблиотеки **movie_review** - платформа, позволяющая людям находить, обозревать и обсуждать различное кино любых жанров и категорий.

Функции:
  =
  + Информация о фильме
  + Категории
  + Жанры
  + Трейлер
  + Кадры из фильма
  + Система аутентификации
  + Отзывы
  + Рейтинг
  + Режиссеры и актеры
  + Фильтры
  
Используемый стек:
   =
   + Python 3.10
   + Django 4, django-ckeditor
   + SQLite
